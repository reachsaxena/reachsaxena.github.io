Cube notations

If you see F it means to move the front face. If you see B it stands for the back. If you see L it stands for the left face. If you see R it stands for the right side. If you see D it means trun the bottom face. And if you see it means to trun the topp face. If you see a prime sing it means trun the face counter clockwise and if you see no prime sign it means to trun the face clockwise. If you see a two at the end, it means the trun the face two times and if you see no 2 it means trun it one time.
